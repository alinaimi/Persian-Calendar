# Persian Calendar
A plug-in for common commander applications to convert the Georgian Calendar to Persian.

Getting the Codes
=============

Checkout (https://github.com/alinaimi/Persian-Calendar)

or

        git clone https://github.com/alinaimi/Persian-Calendar .
